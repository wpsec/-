使用方式：
为了采集信息的准确性，请跟据系统版本选择x86或者x64的Agent进行采集，因为有的采集任务是跟据系统位数进行的。
For the accuracy of the information collected, please select the x86 or x64 Agent to collect according to the system version, because some collection tasks are carried out according to the number of system bits.

Linux Agent：需要先chmod +x 给程序执行权限，并且使用sudo权限运行。
Linux Agent: You need to chmod +x to give the program execution permission first, and run it with sudo permission.
